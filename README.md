## ✎ᝰ. RUNNING

- Clone Repository

```bash
git clone https://github.com/Hunga9k50doker/rise.git
cd rise
```

- Install Dependency

```bash
npm install
```

- Setup config in .env

```bash
nano .env
```

- Setup input value

* proxy: http://user:pass@ip:port

```bash
nano proxies.txt
```

- privatekey: how to get => join my channel: https://t.me/airdrophuntersieutoc

```bash
nano privateKeys.txt
```

